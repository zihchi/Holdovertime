FAA 注意事項 (CAUTIONS) HOT GUIDELINES FOR SAE TYPE IV FLUIDS

▼
• 應用這些數據的責任仍由使用者（機組人員）承擔。
• 在惡劣天氣條件、強降水率或高濕度環境下，防護時間將會縮短。
• 強風、噴射氣流（Jet blast）或吹雪可能會使防護時間低於給定範圍的最低值。
• 當飛機蒙皮溫度低於外部氣溫（OAT）時，防護時間可能會縮短。
• 地面除冰 / 防冰所使用的液體，無法提供飛行中的結冰防護。
• 本表格僅供起飛前計畫使用，且必須與起飛前檢查程序（Pre-takeoff check procedures）配合執行。寬限時間不可藉由目視檢查來延長。
• 當機翼下表面靠近油箱的區域出現霜或冰時，即表示存在「冷浸機翼（Cold-soaked wing）」的狀況。此時的防冰步驟嚴禁使用 50/50 濃度的 Type II 或 Type IV 防冰液，以免發生防冰液結冰的危險。

FAA 規範與備註 (Notes) NOTE for SAE TYPE IV FLUIDS

▼
Note 1 若要使用本表中的防護時間 (HOTs)，請確保所使用的防冰液及稀釋比例已列於「經防冰效能與氣動力接受度測試之第四型防冰液表 (Table 58)」中。必須確認並適用該防冰液的任何使用限制。
Note 2 確保遵守最低操作使用溫度 (LOUT)。當無法使用第四型 (Type IV) 防冰液時，請考慮使用第一型 (Type I) 防冰液。
Note 3 凍霧 (Freezing mist) 最好透過實際觀測來確認。航空例行天氣報告 (METAR) 從不會報告凍霧；然而，當氣溫在 0 °C (32 °F) 或以下且有霧 (Mist) 時，就有可能發生。
Note 4 在冰晶混合凍霧 (Freezing fog) 或凍霧 (Freezing mist) 的情況下，請適用凍霧的防護時間。
Note 5 必須使用「降雪強度與盛行能見度關係表 (Table 54)」來確認降水強度不大於「中度 (Moderate)」。如果回報的能見度對應到「大雪 (Heavy)」等級的降水強度，則無有效的防護時間。
Note 6 若要判定降雪強度，必須使用「降雪強度與盛行能見度關係表 (Table 54)」。
Note 7 在極輕雪 (Very light snow)、小雪 (Light snow) 或中度雪 (Moderate snow) 混合冰晶的情況下，請適用降雪的防護時間。
Note 8 包含輕度、中度與重度凍毛毛雨 (Freezing drizzle)。如果無法確切辨識出是凍毛毛雨，請直接適用「輕凍雨 (Light freezing rain)」的防護時間。
Note 9 這些防護時間適用於「中度 (Moderate)」降水強度。若遇到極輕雪或小雪混合小雨或毛毛雨時，請改用「輕凍雨 (Light freezing rain)」的防護時間。必須使用「降雪強度與盛行能見度關係表 (Table 54)」來確認降水強度。如果回報的能見度對應到「大雪 (Heavy)」等級，則無有效的防護時間。
Note 10 當氣溫在 0 °C (32 °F) 或以下時，此氣候條件無有效的防護時間規範。
Note 11 包含大雪、冰珠、中度與重度凍雨、小冰雹與冰雹 (Table 52 提供了 Type IV EG 防冰液的寬限時間，Table 53 則提供了 Type IV PG 防冰液在冰珠與小冰雹情況下的寬限時間)。
Note 12 當氣溫低於 -10 °C (14 °F) 時，此氣候條件無有效的防護時間規範。
CAUTIONS 適用於上表防護時間之注意事項 (Cautions) 可參閱第 31 頁。
Allowance Note 冰珠/小冰雹相關之寬限時間（Allowance Times）僅適用於 100/0 原液，且飛機的起飛仰轉速度（Rotation Speed）必須達到特定標準（PG基底防冰液需 >100kts）。
