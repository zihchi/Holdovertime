FAA Table 33 規範與備註 (Notes)

Note 1 確保遵守最低操作使用溫度（LOUT）。當無法使用第四型（Type IV）防冰液時，請考慮使用第一型（Type I）防冰液。
Note 2 凍霧（Freezing mist）最好透過實際觀測來確認。航空例行天氣報告（METAR）從不會報告凍霧；然而，當氣溫在 0 °C (32 °F) 或以下且有霧（Mist）時，就有可能發生。
Note 3 在冰晶（Ice crystals）混合凍霧（Freezing fog）或凍霧（Freezing mist）的情況下，請適用「凍霧」的防護時間。
Note 4 必須使用「降雪強度與盛行能見度關係表」（Table 54）來確認降水強度不大於「中度（Moderate）」。如果回報的能見度對應到「大雪（Heavy）」等級的降水強度，則無有效的防護時間。
Note 5 若要判定降雪強度，必須使用「降雪強度與盛行能見度關係表」（Table 54）。
Note 6 在極輕雪（Very light snow）、小雪（Light snow）或中度雪（Moderate snow）混合冰晶的情況下，請適用降雪的防護時間。
Note 7 包含輕度、中度與重度凍毛毛雨（Freezing drizzle）。如果無法確切辨識出是凍毛毛雨，請直接適用「輕凍雨（Light freezing rain）」的防護時間。
Note 8 這些防護時間適用於「中度（Moderate）」降水強度。若遇到極輕雪或小雪混合小雨或毛毛雨時，請改用「輕凍雨（Light freezing rain）」的防護時間。必須使用「降雪強度與盛行能見度關係表」（Table 54）來確認降水強度。如果回報的能見度對應到「大雪（Heavy）」等級，則無有效的防護時間。
Note 9 當氣溫在 0 °C (32 °F) 或以下時，此氣候條件（冷浸機翼降雨）無有效的防護時間規範。
Note 10 包含大雪、冰珠、中度與重度凍雨、小冰雹與冰雹（Table 53 提供了 Type IV PG 防冰液在冰珠與小冰雹情況下的寬限時間）。
Note 11 當氣溫低於 -10 °C (14 °F) 時，此氣候條件無有效的防護時間規範。

FAA 注意事項 (CAUTIONS)

• 應用這些數據的責任仍由使用者（機組人員）承擔。
• 在惡劣天氣條件、強降水率或高濕度環境下，防護時間將會縮短。
• 強風、噴射氣流（Jet blast）或吹雪可能會使防護時間低於給定範圍的最低值。
• 當飛機蒙皮溫度低於外部氣溫（OAT）時，防護時間可能會縮短。
• 地面除冰 / 防冰所使用的液體，無法提供飛行中的結冰防護。
• 本表格僅供起飛前計畫使用，且必須與起飛前檢查程序（Pre-takeoff check procedures）配合執行。
• 當機翼下表面靠近油箱的區域出現霜或冰時，即表示存在「冷浸機翼（Cold-soaked wing）」的狀況。此時的防冰步驟嚴禁使用 50/50 濃度的 Type II 或 Type IV 防冰液，以免發生防冰液結冰的危險。
